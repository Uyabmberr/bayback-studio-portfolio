# Bayback Studio Portfolio

Website portofolio statis untuk karya editing dan desain Bayback.

Live website: https://bayukarya.vercel.app

## Isi Website

- Hero portfolio dengan embed reel Instagram.
- Kanal karya dari akun personal dan brand.
- Panel pilihan saat akun diklik.
- Form brief cepat untuk WhatsApp atau Instagram DM.
- Animasi modern, ringan, dan responsif.

## Deploy

Project ini bisa langsung dideploy ke Vercel sebagai static website. Entry point utama ada di `index.html`.
